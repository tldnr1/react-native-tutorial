2. Text
<Text></Text> 로 사용
numberOfLines={숫자} : Text가 최대 몇줄인지 정함
onPress={함수} : Text가 눌렸을때 반응 지정

 ㄴ 2) 함수형태로 사용 : const handlePress = () => console.log("Text Pressed")
                              <Text onPress={ handlePress }></Text>
